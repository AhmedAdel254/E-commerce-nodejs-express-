# E-commerce Backend API

A fully-featured E-commerce Backend API built with Node.js and MongoDB.
The project focuses on real-world business logic including products, orders, search, and pagination.

## Features
- User Authentication & Authorization (JWT)
- Products & Categories Management
- Cart Management
- Order Management (without payment gateway)
- Search, Filtering & Pagination
- Role-based access control (Admin / User)
- Secure RESTful APIs
- Centralized Error Handling

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Postman

## Project Scope
This project simulates a real-world e-commerce system.
Orders are fully implemented including order creation, order items, and order status.
Payment gateway integration is intentionally excluded to focus on backend architecture and business logic.
